## ✨ Features

### 🕰️ World Clock
- Track multiple time zones at a glance with real-time updates
- Analog & digital clock display with 12H / 24H toggle
- Automatic DST detection with visual indicators
- Full-screen focus mode for distraction-free viewing

### 🎨 Visual Themes
- 6 built-in color skins — Cosmic, Aurora, Rose, Ocean, Ember & more
- Custom background image with adjustable blur & dim intensity
- Dynamic overlay effects: ✨ Stars · 🌠 Meteor · 🍂 Leaves · ❄️ Snow · 🌧️ Rain · ⚡ Thunder · 💗 Hearts · 🌌 Glow

### 🎵 Ambient Sound Player
- 11 ambient tracks — all synthesized natively, zero external dependencies
- White noise: Rain · Heavy Rain · Ocean · Forest · Fireplace · Crickets · Cosmos
- Music: Piano · LoFi · Strings · Dawn Birdsong
- Volume control, loop toggle, animated EQ visualizer

### ⚙️ More
- Time zone converter built-in
- Multi-language support (EN / ZH)
- Custom font & font scale
- All preferences saved locally

## 🛠️ Tech Stack

- Pure **HTML / CSS / JavaScript** — single file, zero dependencies
- **Web Audio API** — all ambient sounds synthesized in-browser
- **Canvas API** — dynamic background animations
- **Electron** — Windows desktop wrapper
